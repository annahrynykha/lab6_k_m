У цьому репозиторії знаходяться три компоненти:
1. **server** - серверна частина (обробка подій, >1024 клієнтів).
2. **client** - клієнтська частина (консольний чат).
3. **test** - утиліта для генерації 1100 з'єднань та тестування навантаження.

## Інструкція із запуску
1. Скомпілюйте всі три проєкти.
2. Першим завжди запускайте `server.exe`.
3. Для ручного тестування запустіть один або кілька `client.exe`.
4. Для перевірки навантаження запустіть `stresstest.exe`.
